# Changelog

Todos los cambios notables en este proyecto serán documentados en este archivo.

El formato está basado en [Keep a Changelog](https://keepachangelog.com/es-ES/1.0.0/),
y este proyecto adhiere a [Semantic Versioning](https://semver.org/lang/es/).

## [1.0.0] - 2026-05-18

### Added
- Sistema de gestión de cargas para camión de transporte
- Módulo de camión con control de peso y capacidad máxima (2500 kg)
- Sistema de validación de peligrosidad para circulación en ruta
- Implementación de 8 tipos de cosas transportables (Knight Rider, Bumblebee, paquete de ladrillos, arena a granel, batería antiaérea, contenedor portuario, residuos radioactivos, embalaje de seguridad)
- Cálculo automático de bultos según tipo y características de cada cosa
- Sistema de consecuencias automáticas al cargar cosas en el camión
- Consultas de peso, peligrosidad y búsqueda de cosas cargadas
- Suite completa de tests para cosas y camión
- Guías de contribución y código de conducta
